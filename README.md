# カンジマスター / Kanji Master

小学生向けの漢字学習WEBアプリケーションです。
1年生から6年生まで、すべての学習漢字を網羅し、読み方、書き順アニメーション、辞書機能、クイズ、マイページ（称号・スコアシステム）を備えています。

## 機能ハイライト
- **学年別漢字ナビゲーション**: 1〜6年生の配当漢字を学年ごとに学習可能。
- **書き順表示＆練習機能**: 筆順アニメーションを見ながら、ブラウザ上で実際に漢字を書く練習ができます（`hanzi-writer` を使用）。
- **クイズモード**: 読み方（音読み・訓読み）、書き取り、対義語などの各種クイズモードを搭載。
- **レベルシステム**: クイズに正解してXPを貯めると、レベルアップして新しい「称号」が手に入ります。
- **にがて帳**: 苦手な漢字をマイページで一覧管理。クリック一つで詳細ページに戻って復習できます。

## 開発環境のセットアップ

Node.js (v18以降) がインストールされている環境で、以下のコマンドを実行します。

```bash
# クローンと移動
git clone https://github.com/your-username/kanji-master.git
cd kanji-master

# 依存モジュールのインストール
npm install

# 開発サーバーの起動（ローカルで動作確認）
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスするとアプリが起動します。

## プロダクションビルドと公開
このリポジトリは GitHub Actions を使用して GitHub Pages への自動デプロイが設定されています。
`main` または `master` ブランチにコードをプッシュするだけで、自動的にビルドされ公開されます。

手動でビルドする場合は以下のコマンドを実行してください。

```bash
npm run build
```

`dist` フォルダに公開用ファイルが生成されます。
